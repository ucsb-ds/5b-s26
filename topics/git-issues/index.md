# Git Issues

This section helps with common Git, GitHub, and Jupyter setup problems.

You do not need to understand every line of the error. Just find the page that matches what you see.

If your screen shows one of these messages, open the matching page:

1. [Problems with the key file](/git-issues/problems-with-the-key-file)
2. [Host key verification failed](/git-issues/host-key-verification-failed)
3. [Key exists, but does not match (recreate SSH key)](/git-issues/key-does-not-match)
